What this app is and how it's built.

Book Catalog is a small Angular front end for browsing and editing a collection of books served by an HTTP API at http://localhost:3000/books.

It is built with:

  • Angular 22 standalone components and signals
  • httpResource for reactive data loading
  • Signal-based forms (@angular/forms/signals)
  • Tailwind CSS v4 for styling
